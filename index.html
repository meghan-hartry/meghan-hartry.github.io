Brown University Course Data

About



This project was created by a team of four University of Iowa students for the first annual Midwest Big Data Hackathon (2016). Our goal was to create an indepth data visualization for the dataset of course data at Brown University. We used D3, Javascript, jsqrl, HTML, and CSS to complete our task.
